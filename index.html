Rogue Genesia Equipment Planner v0.3

Based on decompiled code analysis (Game Version ~0.9.x). Modifier availability per slot is inferred and may have inaccuracies due to missing game data files. Secondary modifier lists show all possibilities; the number of available secondary slots (positive/negative) depends on the chosen Main modifier in-game (this planner uses default counts from C# unless the main modifier data specifies otherwise - FIX #1).
Required Soul Level is an ESTIMATE based on modifier rarity and tier/level scaling (FIX #2, #3), as the exact base cost values are defined in game data assets not available here (Inaccuracy #5). Custom modifier rarities/levels are also estimations. Calculations aim to replicate game logic (FIX #4) but may have discrepancies.

Planner created based on provided C# source code.
