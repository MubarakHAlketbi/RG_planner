Rogue Genesia Equipment Planner v0.2

Based on decompiled code analysis (Game Version ~0.9.x). Modifier availability per slot is inferred and may have inaccuracies due to missing game data files. Secondary modifier lists show all possibilities; actual availability and the number of secondary slots depend on the chosen Main modifier in-game (this planner uses default counts of 2 positive, 1 negative unless the main modifier dictates otherwise).
Required Soul Level is an ESTIMATE based on modifier rarity and tier/level scaling, as the exact values are defined in game data assets not available here. Custom modifier rarities/levels are also estimations. Calculations aim to replicate game logic but may have discrepancies.

Planner created based on provided C# source code. Improvements added: Tier/Level inputs, Selected Rarity Indicator, Unique Modifier Indicator, Required Soul Level calculation (estimated), God Stone calculations updated, Tier map extended.
